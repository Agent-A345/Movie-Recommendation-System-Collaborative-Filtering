## 🎥 Movie Recommendation System using Collaborative Filtering

A machine learning project built using Python and Jupyter Notebook that recommends movies based on user ratings and viewing preferences. The project utilizes collaborative filtering techniques to identify similar users, discover viewing patterns, and generate personalized movie recommendations using the MovieLens dataset.

## 🚀 Features
* **Collaborative Filtering Engine** – Generate recommendations using user rating behavior.
* **Personalized Movie Suggestions** – Recommend movies tailored to user preferences.
* **User Similarity Analysis** – Identify users with similar viewing patterns.
* **Movie Similarity Detection** – Discover related movies based on audience behavior.
* **Rating-Based Recommendation System** – Utilize historical ratings for predictions.
* **Data Cleaning & Preprocessing** – Prepare MovieLens datasets for analysis.
* **Exploratory Data Analysis (EDA)** – Analyze user engagement and rating trends.
* **Recommendation Score Generation** – Rank movies based on predicted relevance.
* **Scalable Recommendation Workflow** – Build reusable recommendation pipelines.
* **Real-World MovieLens Dataset** – Practice recommendation system development using industry-standard data.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/Movie-Recommendation-System-Collaborative-Filtering.git
```
2. **Navigate to the project folder**:
```
cd Movie-Recommendation-System-Collaborative-Filtering
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter scikit-learn
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
movie_recommendation_system.ipynb
```
